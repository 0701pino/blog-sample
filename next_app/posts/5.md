# 見出しと水平線の作成方法

## 見出し

Markdownで見出しを作成するには、`#`記号を使用します。

```markdown
# 見出し1

## 見出し2

### 見出し3

#### 見出し4

##### 見出し5

###### 見出し6
```

## 水平線

Markdownで水平線を作成するには、`---`または`***`を使用します。

```markdown
---
```
